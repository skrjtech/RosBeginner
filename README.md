# ROS Versions 
Ros1, Ros2のインストール用パッケージ\
インストールしたいバージョンをmakeで簡単にインストールが可能
```
git clone https://github.com/skrjtech/ros-versions.git
cd ros-versions
```
# Ros1 Version インストール方法
記述例
```
cd Ros1
make rosVersion
```
### melodic
```
make melodic-cpu
```
### melodic gpu
```
make melodic-gpu
```
### neotic
```
make neotic-cpu
```
### neotic gpu
```
make neotic-gpu
```
# Ros2 Version インストール方法
記述例
```
cd Ros2
make rosVersion
```
